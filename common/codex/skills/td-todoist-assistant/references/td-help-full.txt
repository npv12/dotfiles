## td
Usage: td [options] [command]

Todoist CLI

Options:
  -V, --version            output the version number
  --no-spinner             Disable loading animations
  --progress-jsonl [path]  Output progress events as JSONL to stderr or file
  -v, --verbose            Increase output verbosity (repeat up to 4x: -v, -vv,
                           -vvv, -vvvv)
  -h, --help               display help for command

Commands:
  add                      Quick add task with natural language (e.g., "Buy milk
                           tomorrow p1 #Shopping")
  today                    Show tasks due today and overdue
  upcoming                 Show tasks due in the next N days (default: 7)
  inbox                    List tasks in Inbox
  completed                Show completed tasks
  task                     Manage tasks
  project                  Manage projects
  label                    Manage labels
  comment                  Manage comments
  section                  Manage project sections
  workspace                Manage workspaces
  activity                 View activity logs
  reminder                 Manage task reminders
  settings                 Manage user settings
  auth                     Manage authentication
  stats                    View productivity stats and karma
  filter                   Manage filters
  notification             Manage notifications
  skill                    Manage coding agent skills/integrations
  completion               Manage shell completions
  help [command]           display help for command

Note for AI/LLM agents:
  Use "td task add" (not "td add") to create tasks with structured flags.
  Use --json or --ndjson flags for unambiguous, parseable output.
  Default JSON shows essential fields; use --full for all fields.

Usage: td add [options] [text]

  --date-format <format>            Date format: us (MM-DD) or intl (DD-MM)
                                    (choices: "us", "mm-dd-yyyy", "mdy", "intl",

  --start-day <day>                 Week start: monday, tuesday, etc. (choices:

  --auto-reminder <min>             Default reminder minutes (0 to disable)
  --next-week <day>                 "Next week" day: monday, tuesday, etc.
                                    (choices: "monday", "mon", "tuesday", "tue",

                                    "sunday", "sun")
  --start-page <page>               Default view: inbox, today, or project URL
  --reminder-push <bool>            Push reminders: on/off (choices: "on",
                                    "off", "true", "false", "yes", "no", "1",

  --reminder-desktop <bool>         Desktop reminders: on/off (choices: "on",
                                    "off", "true", "false", "yes", "no", "1",

  --reminder-email <bool>           Email reminders: on/off (choices: "on",
                                    "off", "true", "false", "yes", "no", "1",
